gmsh_msh1_element_type

type, public :: gmsh_msh1_element_type

Derived type to for reading the n-th element in the Gmsh MSH file format version 1

Warning

  • The elm_number must be a positive (non-zero) integer.
  • The reg_phys must be a positive integer, or zero. If reg_phys is equal to zero, the element is considered not to belong to any physical entity.
  • The reg_elem must be a positive (non-zero) integer.

Note

The elm_number do not necessarily have to form a dense nor an ordered sequence.
