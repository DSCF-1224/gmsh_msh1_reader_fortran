gmsh_msh1_node_number_type

type, public :: gmsh_msh1_node_number_type

Derived type to for reading the number (index) of the n-th node in the mesh.

Warning

The number must be a positive (non-zero) integer.

Note

The number do not necessarily have to form a dense nor an ordered sequence.
